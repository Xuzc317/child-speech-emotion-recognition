# 项目总结
## Project Summary

## 项目背景

本项目为本科毕业设计/科研实践项目，主题为**儿童语音情绪识别** (Children Speech Emotion Recognition)。基于C-BESD儿童双语情绪语音数据集，研究针对儿童语音特点的情绪识别模型。

儿童语音与成人语音在音高、语速、发音清晰度等方面存在显著差异，需要专门的声学特征分析和模型适配。本项目旨在探索适合儿童语音情绪识别的技术方案。

## 任务定义

**6类情绪识别分类任务**：
1. **ANGER** (愤怒)
2. **HAPPY** (快乐)
3. **SAD** (悲伤)
4. **DISGUST** (厌恶)
5. **NEUTRAL** (中性)
6. **FEAR** (恐惧)

**评估指标**：准确率 (Accuracy) 和 Macro F1分数。

## 数据集

### C-BESD (Children's Basic Emotion Speech Dataset)
- **来源**：专业录制的儿童双语情绪语音
- **规模**：原始语音约4,320条，数据增强后约12,540条
- **说话人**：70名儿童 (35男，35女)，年龄6-12岁
- **语言**：泰卢固语和英语双语
- **划分**：说话人无关划分，确保模型泛化能力
- **特点**：儿童非专业演员，专业录音环境，6种基本情绪

**重要说明**：本GitHub仓库不包含原始数据集文件，需联系数据集所有者获取。

## 技术方法

### 特征工程
提取94维融合特征：
- **MFCC**：40维，Mel频率倒谱系数
- **Mel Spectrogram**：40维，梅尔频谱
- **Chroma STFT**：12维，色度特征
- **ZCR**：1维，过零率
- **RMS**：1维，均方根能量

### 数据增强
提高模型鲁棒性的三种技术：
1. **加噪** (Additive noise)
2. **时长拉伸** (Time stretching)
3. **音高偏移** (Pitch shifting)

### 模型架构

#### 核心模型：DrseCNN (Deep Residual Squeeze-Excitation CNN)
- **多阶段卷积结构**：分层特征提取
- **ResSE Block**：残差连接 + SE注意力机制
- **SE注意力**：通道注意力，增强重要特征
- **自适应池化**：处理变长输入
- **全连接分类器**：最终情绪分类

#### 对比模型
- **CNN**：标准卷积神经网络
- **BiLSTM**：双向长短时记忆网络
- **Transformer**：自注意力模型
- **CNN-BiLSTM**：卷积与循环网络结合
- **SigWavNet**：信号处理网络

## 实验结果

### 主要性能指标
| 模型 | 最佳准确率 | Macro F1 | 参数量 |
|------|------------|----------|--------|
| DrseCNN (Full) | 0.8599 | 0.8506 | ~45.5M |
| CNN | 0.7214 | - | ~5.8M |
| SigWavNet | 0.7318 | - | ~16.9M |
| BiLSTM | 0.7226 | - | ~12.1M |
| Transformer | 0.6948 | - | ~2.0M |

### 重要说明
**复现性说明**：早期实验记录的最佳准确率约为0.86，但由于当时未固定随机种子，该结果可能无法完全稳定复现。当前仓库以可复现/可说明的**0.8599准确率**作为主要报告结果。

### 消融实验结果
实验验证了以下组件的有效性：
1. **数据增强**：加噪、拉伸、变调均提升泛化能力
2. **特征融合**：多特征组合优于单一特征
3. **模型组件**：SE注意力和残差连接均有显著贡献

## 项目意义与局限

### 贡献
1. **儿童语音专门化**：针对儿童语音特点设计特征和模型
2. **模型创新**：提出适用于儿童语音的DrseCNN架构
3. **实验完整**：系统的对比实验和消融分析
4. **文档完善**：详细的技术文档和实验记录

### 局限性
1. **数据集规模**：儿童情绪语音数据相对稀缺
2. **复现性**：早期实验随机性控制不足
3. **计算资源**：部分模型训练需要较多GPU资源
4. **应用范围**：学术研究性质，非生产部署级别

## 使用说明

### 代码运行前提
1. **数据集获取**：需联系C-BESD数据集所有者
2. **环境配置**：安装requirements.txt中的依赖包
3. **硬件要求**：GPU内存建议8GB以上

### 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 准备数据 (需自行获取)
# 3. 运行训练 (示例)
python src/training/train.py
```
