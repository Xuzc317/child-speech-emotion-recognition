# 数据集说明
## Dataset Documentation

## 概述

本项目使用**C-BESD** (Children's Basic Emotion Speech Dataset) 数据集，这是一个专门为儿童语音情绪识别研究设计的数据集。

**重要声明**: 本GitHub仓库**不包含**原始数据集文件。数据集需从官方渠道获取，请遵守相关使用协议。

## 数据集基本信息

### 核心属性
| 属性 | 说明 |
|------|------|
| **名称** | C-BESD (Children's Basic Emotion Speech Dataset) |
| **别名** | BESD (Bilingual Emotion Speech Dataset for Children) |
| **语言** | 泰卢固语和英语双语 |
| **情绪类别** | 6类 (ANGER, DISGUST, FEAR, HAPPY, NEUTRAL, SAD) |
| **说话人数量** | 70名儿童 (35男，35女) |
| **年龄范围** | 6-12岁 |
| **录制环境** | 专业录音室 |
| **录制方式** | 儿童非专业演员，指导性情绪表达 |

### 数据规模
| 统计项 | 数量 |
|--------|------|
| **原始语音条数** | 约4,320条 |
| **每名儿童每情绪** | 5条语句 |
| **数据增强后** | 约12,540条 (原始×3) |
| **总时长** | 约18小时 (原始) |

### 情绪分布
| 情绪类别 | 英文标签 | 类别ID | 原始样本数 | 比例 |
|----------|----------|--------|------------|------|
| 愤怒 | ANGER | 0 | ~720 | 16.7% |
| 厌恶 | DISGUST | 1 | ~720 | 16.7% |
| 恐惧 | FEAR | 2 | ~720 | 16.7% |
| 快乐 | HAPPY | 3 | ~720 | 16.7% |
| 中性 | NEUTRAL | 4 | ~720 | 16.7% |
| 悲伤 | SAD | 5 | ~720 | 16.7% |

## 数据特点

### 儿童语音特性
1. **音高较高**: 儿童声带较短，基频高于成人
2. **语速变化大**: 情绪表达时语速变化明显
3. **发音清晰度**: 年龄较小儿童可能存在发音不清晰
4. **情绪表达**: 儿童情绪表达相对直接和强烈

### 录制质量
- **采样率**: 16kHz (标准语音识别采样率)
- **位深度**: 16-bit
- **格式**: WAV格式
- **噪声控制**: 专业录音室环境，背景噪声较低

### 标注信息
- **情绪标签**: 专业标注员标注，经过一致性检验
- **说话人信息**: 性别、年龄分组信息
- **文本转录**: 部分语句提供文本转录
- **质量评分**: 部分数据有录音质量评分

## 数据预处理

### 原始数据文件结构
```
BESD/
├── ANGER/
│   ├── child01_anger_001.wav
│   ├── child01_anger_002.wav
│   └── ...
├── DISGUST/
├── FEAR/
├── HAPPY/
├── NEUTRAL/
└── SAD/
```

### 预处理步骤
1. **音频加载**: 使用librosa加载，统一采样率
2. **静音修剪**: 去除首尾静音段
3. **长度标准化**: 裁剪或补零至固定时长(2.5秒)
4. **归一化**: 音量归一化处理
5. **特征提取**: 提取94维声学特征

## 数据增强策略

为应对数据量有限的问题，采用三种数据增强技术：

### 1. 加噪 (Additive Noise)
- **方法**: 添加高斯白噪声
- **强度**: 噪声幅度为原始信号最大值的0-3.5%
- **目的**: 提高模型对背景噪声的鲁棒性

### 2. 时长拉伸 (Time Stretching)
- **方法**: librosa.effects.time_stretch
- **拉伸率**: 0.8倍 (20%加速)
- **目的**: 模拟语速变化

### 3. 音高偏移 (Pitch Shifting)
- **方法**: librosa.effects.pitch_shift
- **偏移量**: 0.7个半音
- **目的**: 模拟音高变化

**增强效果**: 每个原始样本生成3个增强样本 (原始+加噪+拉伸变调)

## 数据集划分

### 划分原则
- **说话人无关**: 训练集和测试集的说话人不重叠
- **性别平衡**: 保持男女比例均衡
- **年龄分布**: 各年龄组在训练测试中均有代表
- **情绪平衡**: 各类情绪样本比例一致

### 划分比例
| 数据集 | 说话人比例 | 样本比例 | 用途 |
|--------|------------|----------|------|
| **训练集** | 70% (49人) | 70% | 模型训练 |
| **验证集** | 15% (10人) | 15% | 超参调优 |
| **测试集** | 15% (11人) | 15% | 最终评估 |

### 划分实现
```python
# 代码示例 (简化)
speakers = list(range(1, 71))  # 70个说话人
train_spk = speakers[:49]      # 训练说话人
val_spk = speakers[49:59]      # 验证说话人
test_spk = speakers[59:]       # 测试说话人
```

## 特征提取

### 特征维度总览
| 特征类型 | 维度 | 描述 |
|----------|------|------|
| **MFCC** | 40 | Mel频率倒谱系数，反映声道特性 |
| **Mel Spectrogram** | 40 | 梅尔频谱，模拟人耳听觉特性 |
| **Chroma STFT** | 12 | 色度特征，反映音高类别分布 |
| **ZCR** | 1 | 过零率，反映语音清浊特性 |
| **RMS** | 1 | 均方根能量，反映语音强度 |
| **总计** | 94 | 多特征融合 |

### 提取参数
| 参数 | 值 | 说明 |
|------|-----|------|
| **采样率** | 16000 Hz | 标准语音采样率 |
| **FFT窗口** | 512 | 傅里叶变换窗口大小 |
| **Hop长度** | 160 | 帧移，10ms |
| **Mel滤波器数** | 40 | MFCC和Mel频谱维度 |
| **MFCC阶数** | 40 | 保留全部40阶系数 |
