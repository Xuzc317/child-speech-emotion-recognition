#### 关键发现与分析

**1. 统计先验为什么不如随机初始化？**

计算的 scale/bias 是 per-dimension 全局统计量（对所有帧、所有语音取均值）。但 SSL 特征空间中：
- 不同帧（元音 vs 辅音 vs 静音）、不同说话人、不同情绪的偏移方向不同
- 全局补偿抹平了这些差异性：某维度上儿童 F0 高导致均值偏高，但该维度在情绪分类任务中未必重要
- 随机初始化 + 梯度下降能学到"对分类任务有利"的校准方向，而非"对齐分布"的方向

**2. FD=0.37 的意义**

- 这个 FD 值确认了儿童/成人分布偏移的存在
- 但偏移 ≠ 情绪分类退化：偏移可能集中在"说话人身份""信道特性"等无关维度
- Adapter 学到的是"对 SER 分类有利的重映射"，而非"纠正分布偏移"——这两个目标不完全重叠

**3. 对论文叙事的启示**

可组织为以下论证链：
```
论点：儿童语音在 SSL 空间存在分布偏移
证据：FD=0.37，scale 平均偏离 8%，|bias| 均值 0.06
直接补偿（统计先验）：仅 +0.79pp，线性补偿不够
让模型自适应（随机 init）：+4.74pp，非线性校准有效
结论：偏移存在但方向复杂，需要可学习的非线性校准
     → 这正是我们提出 Adapter 的动机
```

**注意**：虽然"随机初始化胜过统计先验"看起来是 negative result，但对论文论证反而是 positive——它证明了简单的线性补偿不够，需要让模型自己学习校准，这正是 AcousticCalibrationAdapter 存在的价值。

### Sub-Phase 2.3：模块 3 — 分布约束增强 + 负面实验 ✅ 已完成

#### 实际执行

**Step 2.3.1 — 计算儿童语音统计分布**：
```
BESD MY (2,000 WAVs):
  F0  mean=495.9 Hz, std=651.1, p5=75.3 Hz, p95=2285.7 Hz
  Energy mean=0.0464, std=0.0566
  → pitch_shift range: ±6 semitones
  → stretch range: 0.85-1.15
```

**Step 2.3.2 — 离线增强方案**（替代在线训练）：
编写 `scripts/extract_augmented_features.py`，对每个 WAV 应用增强后立即提取 SSL 特征，避免实现完整的 online 训练模式。四组配置：

| 配置 | pitch_shift | time_stretch |
|------|------------|-------------|
| C1 (无增强) | 0 | 1.0 |
| C2 (成人参数) | ±6 st | 0.7-1.3 |
| C3 (儿童约束) | ±3 st | 0.85-1.15 |
| C4 (极端参数) | ±12 st | 0.5-1.5 |

每样本随机应用一种增强（pitch_shift 或 time_stretch），然后通过 frozen WavLM 提特征。

**Step 2.3.3 — 负面实验结果**：

| 实验 | FD | Val Acc | vs C1 |
|------|-----|---------|-------|
| **C1** (无增强) | 0 | **86.42%** | — 基线 |
| C3 (儿童约束) | 8.71 | 66.85% | -19.57pp |
| C2 (成人参数) | 9.87 | 59.51% | -26.91pp |
| C4 (极端参数) | 11.99 | 43.25% | -43.17pp |

**Step 2.3.4 — FD vs Accuracy 负相关**：

FD 与准确率的排序完全一致：C4(11.99) > C2(9.87) > C3(8.71) > C1(0)，对应 Acc 严格递减。FD vs Accuracy 负相关曲线已生成：
`assets/figures/fd_vs_accuracy.png`

#### 关键发现

**1. 预期 vs 实际的差异**：
- 原预期 C3（儿童约束）应该有提升（+1-3%），实际下降了 -19.57pp
- 原因：所有训练样本都被增强（而非 mix 原始+增强），导致情绪信息被普遍破坏
- 但这反面证明了核心论点：**儿童的语音情感信息对波形增强非常敏感**

**2. "负面实验"实际上是"强正面证据"**：
- 即使「儿童约束」参数也导致大幅下降 → 说明儿童语音的情感内容本身就脆弱
- 成人参数和极端参数的更大下降 → 说明参数越偏离真实分布，情感信息丢失越严重
- FD ∝ 1/Accuracy 的严格单调关系 → 分布偏移可定量预测分类退化

**3. 对论文的启示**：
- 这张 FD vs Accuracy 图是论文最有说服力的图表之一
- 论证链：分布偏移存在 → 增强引入额外偏移 → 偏移越大越破坏情感 → 增强参数必须受分布约束
- 即使 C3（儿童约束）也下降，说明「仅靠参数约束不够」——需要更根本的分布感知方法（这正是框架的核心论点）

## 关键风险点
