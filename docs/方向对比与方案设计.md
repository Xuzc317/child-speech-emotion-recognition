## 一、核心发现：当前项目 30-40% 准确率的根因

所有模型在 **没有时间维度的数据上做时序建模**。

```
WAV (2.5s) → librosa 帧级特征 → np.mean(所有帧) → (162,) → Conv1d
```

- Conv1d kernel 在 `[ZCR(1), Chroma(12), MFCC(20), RMS(1), Mel(128)]` 的拼接特征上滑动，而非时间轴
- Transformer 输入 (B, 1, 162) → T=1 → attention 无意义
- CRNN 的 LSTM 把下采样后的 ~40 个"位置"当时间步 —— 位置 i 到 i+1 是不同特征组的边界

原始论文 86.82% 完全来自 speaker 重叠导致的数据泄露。去掉泄露后，30-40% 是 162 维均值向量本身能携带的情绪信息上限。

## 三、"分布分析驱动三模块"框架

### 核心叙事

```
儿童语音 ≠ 成人语音（定量统计分布分析）
         ↓
分布偏移在 SER 三个环节的具体表现
         ↓
每个环节对应一个技术模块（被"逼出来的"设计）
         ↓
分类器用 DrseNet（不是核心创新，只是自然汇聚的结果）
```

### 三个技术模块

#### 模块 1：分布校准适配器（Distribution Calibration Adapter）

**问题**：emotion2vec/wav2vec 在成人数据上预训练，儿童语音在其隐空间中存在系统性分布偏移。

**方法**：在 frozen SSL 之上加一个轻量适配器，显式做分布对齐，而非简单 full fine-tune。

**创新点**：面向儿童声学分布偏移设计的适配器结构（不是通用 LoRA）。

#### 模块 2：时序重要性池化（Temporal Importance Pooling）

**问题**：儿童语音的韵律变异性比成人更大（F0 range 更宽、语速波动更大），帧级情感信息密度不均匀。原有 mean pooling 完全丢失此信息。

**方法**：用儿童语音的韵律特征（F0 变异性、语速）调节帧级 attention，实现自适应池化。

**创新点**：池化策略本身由儿童语音分布特性驱动，而非通用 attention pooling。

#### 模块 3：分布约束增强 + 负面实验（Distribution-Constrained Augmentation）

**问题**：成人语音上有效的增强参数（如 pitch shift ±4 semitones）应用到儿童语音上会使其脱离真实分布，破坏情感相关特征。

**方法**：
- 形式化定义儿童语音的合法增强空间 S_child
- 对比实验：无约束增强 → 样本漂出分布（用 Frechet Distance / MMD 衡量）→ 准确率下降
- 分布约束增强（参数由儿童统计分布估计）→ 保持分布内 → 提升

**创新点**：不做新增强方法，而是做"增强空间约束"这个被忽视的问题 + 定量验证。

### 论文结构

```
1. Introduction：儿童语音 ≠ 成人语音，现有 SER 忽视分布偏移
2. Statistical Analysis of Child-Adult Acoustic Divergence
   - F0、formant、语速、韵律变异性的定量对比
   - SSL 隐空间中儿童 vs 成人分布可视化（t-SNE/UMAP）
3. Distribution-Calibrated Framework
   3.1 Distribution Calibration Adapter
   3.2 Temporal Importance Pooling
   3.3 Distribution-Constrained Augmentation
   3.4 Classifier: DrseNet（一句话：这些模块自然汇聚成 DrseNet）
4. Experiments
   - Negative experiment：成人默认配置在儿童数据上失败
   - Ablation：每个模块的独立贡献
   - Cross-language：English → Telugu 迁移
5. Related Work & Conclusion
```
